# Introduzione

MarconiCraft è il server Minecraft degli studenti del Marconi.

> [!important] Importante
> Description
> Attualmente è in **beta**. Questo significa che:
>
> - molte funzionalità devono ancora arrivare
> - alcune cose potrebbero non funzionare
> - il mondo può essere **resettato in qualsiasi momento**
> - il server potrebbe andare giù senza preavviso
>
> Trattatelo con la delicatezza con cui trattereste un Raspberry Pi sotto sforzo.

## Come entrare?

Segui il **[Tutorial](/marconicraft/tutorial.md)** per configurare tutto correttamente.

Se hai dubbi esistenziali, leggi le **[FAQ](/marconicraft/faq.md)**.
