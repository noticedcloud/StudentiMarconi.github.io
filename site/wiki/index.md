# Marconi Wiki

Marconi Wiki è il nostro tentativo di creare una **raccolta di guide fatte dagli studenti, per gli studenti**.

Il progetto è hostato su questo sito.  
Il codice è disponibile pubblicamente su GitHub e pubblicato tramite GitHub Pages.  
È scritto in [Markdown](https://www.markdownguide.org/getting-started/). Niente HTML. Prego.

## Cos'è?

È una raccolta di pagine su argomenti scolastici:

- spiegazioni chiare
- esempi concreti
- riassunti fatti bene
- zero fuffa
- zero copia-incolla dal libro
- zero ChatGPT (opzionale)

Ogni pagina dovrebbe essere:

- **concisa ma completa**
- leggibile in una sera prima della verifica
- abbastanza chiara da salvare una media

## Chi può scrivere?

In teoria chiunque.

In pratica:

- Se hai capito davvero un argomento,
- se sai spiegarlo senza fare copia-incolla,
- se non scrivi "raga è facilissimo" e poi sparisci

...sei perfetto.

Se vuoi contribuire, visita pure la [pagina apposita](/wiki/contributing.md).

## Cosa NON è

- Non è un sito per caricare 200 pagine di appunti grezzi.
- Non è un dump di PDF.
- Non è Wikipedia 2.
- Non è un posto per fare il fenomeno.

È una raccolta di guide fatte bene.

Se una pagina non aiuta davvero qualcuno a capire un argomento, non serve.
